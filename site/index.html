GitHub Pages preview

Welcome to the Wack-Man homepage

This repository now ships with an automatic GitHub Pages deployment so you can preview the project instantly after each push. Drop your game build or documentation into the site/ folder, and it will be published for you.

What to do next

  • Customize the page: Replace this file at site/index.html with your own content.
  • Trigger a deploy: Push changes to the work branch to kick off the GitHub Actions workflow.
  • Share the link: After the first successful run, your page will be live at the repository's GitHub Pages URL.
Need help? Check the repository's Actions tab to monitor the Deploy static content to Pages workflow and find the published link after the deploy finishes.
